# sca-project
This repository contains a complete implementation of the Sine Cosine Algorithm (SCA) developed for an academic project on evolutionary computation and optimization techniques.

The goal of this project is to:
   Explain the principles behind the SCA metaheuristic
   Implement the algorithm in Python
   Test it on a benchmark function (Sphere function)
   Provide visualizations of convergence
   Generate AI-powered images and videos for a full presentation
